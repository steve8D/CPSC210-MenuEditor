# My Online Bakery Store

## An introduction

This is a Java application where you can log in as a manager of an online bakery store 
to make extra money from your interest in baking. You can also log in as a customer to order the items
on the application.


With this application, as a manager/shop owner, you can:

- Add and remove items
- Edit prices and quantity to items on the menu

As a customer, the person can: 

- Add the items into the cart
- Check out with the items on the cart

## User Stories
- As a user (both manager and customer), I want to be able to view the items in different categories on the menu
- As a manager/shop owner, I want to be able to add or remove the items on my menu
- As a manager/shop owner, I want to be able modify an item's price and quantity on my menu.
- As a customer, I want to be able to select the items in different categories and add it to my order list.
