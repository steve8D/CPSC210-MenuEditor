# My Online Bakery Store

## An introduction

This is a Java application where you manage an online bakery store 
to make extra money from your interest in baking. Since you receive external validation 
compliments from your friends and family, you want to show the world your capability
to make an appearance on the national baking competition.


With this application, you can:

- Add, edit, and remove items
- Set prices to items on the menu
